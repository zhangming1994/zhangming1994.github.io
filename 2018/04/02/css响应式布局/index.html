css响应式布局

2018-04-02

加入元标签

响应式布局第一步是加入如下元标签
<meta name="viewport" content="width=device-width,initial-scale=1.0,maximum-scale=1.0,minimum=1.0,user-scalable=no"/>

用法

自适应网页设计的核心是css引入的media query模块,意思是自动探测屏幕宽度，然后加载相应的css文件

加载方式:
max-device-width:400px;意思是设备屏幕的最大输出宽度

1.<link rel="stylesheet" type="text/css" media="screen and(min-width:400px) and (max-device-width:600px)" href="xxx.css"/>
也可以在中间使用逗号表示并列:
media="handheld and (max-width:480px), screen and (min-width:960px)"

//屏幕宽度在400到600像素之间，加载xxx.css文件

2.除了用html标签加载css文件，还可以在现有的css文件中间加载
@import url("xxx.css") screen and(max-device-width:400px);

3.同一个css文件中间，也可以根据不同的分辨率，应用不同的css规则
@media screen and(max-device-width:400px){
    .column{
        float:none;
    }
}
and是关键词，不用解释就应该清楚，类似的还有not，only

图片的自适应

图片的自适应只需要一行代码:im{max-width:100%;},对于大多数嵌入网页的视频也是有效的
老版本的IE不支持max-width,可以使用img{width:100%}
